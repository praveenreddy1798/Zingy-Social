## Zingy Social

Zingy Social is a social application to connect with people around the world.

## Application Links

[Zingy Social Live](https://zingy-social-app.netlify.app/)

## Features

### P0

- Create a Post
  - Like Post
  - Comment & Replies
- Edit & Delete Post
- User Feed
  - Sort by Date, Trending & Recent
- User Profile
  - Add Profile Picture
  - Bio
  - Portfolio URL
  - Follow/Unfollow
- Explore Page
- Bookmark Post
- Authentication
  - Sign-up
  - Login
  - Logout

### P1

- Upload Images with Posts
- Search for post or user
- Loading & Alerts
